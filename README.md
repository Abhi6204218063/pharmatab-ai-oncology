PharmaTab AI Oncology Platform

AI-driven computational oncology platform for tumor evolution modeling, resistance prediction, and personalized therapy optimization using genomic mutation data.

---

Overview

PharmaTab is a research-oriented computational oncology system designed to simulate tumor progression and evaluate targeted therapy strategies using genomic mutation profiles.

The platform integrates multiple AI and simulation engines including tumor evolution modeling, drug resistance prediction, digital patient twins, and survival analysis.

---

Key Features

- Genomic mutation analysis
- Targeted therapy mapping
- Tumor evolution simulation
- Drug resistance prediction
- Digital patient twin modeling
- Virtual patient cohort simulation
- Kaplan-Meier survival prediction
- AI therapy optimization
- Automated clinical-style treatment report generation

---

Platform Architecture

The PharmaTab system consists of several modular engines:

- Mutation Analysis Engine
- Therapy Mapping Engine
- Tumor Evolution Simulator
- Drug Resistance Engine
- Digital Patient Twin Model
- Virtual Cohort Simulator
- Kaplan-Meier Survival Engine
- AI Therapy Optimizer
- Clinical Report Generator

---

Input Data

PharmaTab accepts patient genomic mutation data in:

- CSV format
- TSV format
- TCGA mutation datasets

Example mutation list:

BRAF
NRAS
TP53
PIK3CA

---

Output

The platform generates:

- Tumor growth simulations
- Drug resistance predictions
- Therapy comparison graphs
- Kaplan-Meier survival curves
- Clinical treatment strategy report (PDF)

Installation

Clone the repository

git clone https://github.com/Abhi6204218063/pharmatab-ai-oncology.git

Navigate into the directory

cd pharmatab-ai-oncology

Install dependencies

pip install -r requirements.txt

Run the platform

streamlit run pharmatab_dashboard.py
